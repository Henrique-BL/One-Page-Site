One page site

Ideal para negócios de pequeno e médio porte, o one page é um site que você coloca toda informação que o usuário precisa de forma objetiva e criativa em uma única página.

Wireframe

Wireframe é a representação visual da estrutura básica de uma página. Ele deve ser feito da maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escanei o QR CODE para acessar o Wireframe dessa página.
